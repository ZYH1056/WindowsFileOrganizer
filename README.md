# WindowsFileOrganizer
 适配 Windows 的本地图形化文件整理工具，全程离线运行，不会上传任何本地数据，保护文件隐私。 简约 GUI 界面，分区清晰，新手零门槛上手，支持浅色 / 深色模式切换，高危操作带二次确认弹窗。 核心功能： 扫描桌面、磁盘、U 盘，按后缀 / 关键词自动分类归档文件； 哈希比对查找重复文件，可视化批量清理冗余文件； 可视化批量重命名，实时预览，自动过滤 Windows 非法文件名； 一键规整桌面文件，自动归档杂乱文件，保留快捷方式； 文件批量打包 ZIP，本地保存操作日志，支持撤回误操作。 单文件 exe，无需额外依赖，双击即可运行。项目采用 MIT 协议开源，欢迎提交 Issue 反馈问题、PR 拓展功能。
